Cài n8n bằng Docker (khuyến nghị)
Terminal — Docker ComposeSetup
# 1. Tạo thư mục dự án
mkdir n8n-labs && cd n8n-labs

# 2. Tạo file docker-compose.yml
cat > docker-compose.yml << 'EOF'
version: '3.8'
services:
  n8n:
    image: n8nio/n8n:latest
    ports:
      - "5678:5678"
    environment:
      - N8N_BASIC_AUTH_ACTIVE=true
      - N8N_BASIC_AUTH_USER=admin
      - N8N_BASIC_AUTH_PASSWORD=password123
      - GENERIC_TIMEZONE=Asia/Ho_Chi_Minh
      - N8N_ENCRYPTION_KEY=your-secret-key-here
    volumes:
      - n8n_data:/home/node/.n8n
volumes:
  n8n_data:
EOF

# 3. Khởi động
docker compose up -d

# 4. Mở trình duyệt: http://localhost:5678

Xử lý response — Split Out Node

Sau HTTP Request, response là 1 item chứa array items[]. Dùng Split Out Node để tách thành nhiều items riêng lẻ, mỗi item là 1 repo.

Các query SQL phổ biến trong n8n
PostgreSQL Node — Common QueriesSQL
-- INSERT đơn hàng mới
INSERT INTO orders (customer_id, amount, status, created_at)
VALUES (
  {{ $json.customerId }},
  {{ $json.amount }},
  'pending',
  NOW()
) RETURNING id;

-- UPSERT sản phẩm (thêm mới hoặc update nếu đã tồn tại)
INSERT INTO products (sku, name, stock)
VALUES ({{ $json.sku }}, {{ $json.name }}, {{ $json.stock }})
ON CONFLICT (sku) DO UPDATE SET
  stock = EXCLUDED.stock,
  updated_at = NOW();

-- Query báo cáo doanh thu
SELECT
  DATE(created_at) as date,
  COUNT(*) as total_orders,
  SUM(amount) as revenue
FROM orders
WHERE created_at >= NOW() - INTERVAL '7 days'
GROUP BY DATE(created_at)
ORDER BY date DESC;

Code Node — Patterns thực tếJavaScript
// 1. Làm việc với nhiều items
const results = $input.all().map(item => ({
  json: {
    fullName: `${item.json.firstName} ${item.json.lastName}`,
    email: item.json.email.toLowerCase(),
    ageGroup: item.json.age >= 18 ? 'adult' : 'minor'
  }
}));
return results;

// 2. Aggregate — Tính thống kê
const items = $input.all();
const amounts = items.map(i => i.json.amount);
const total = amounts.reduce((a,b) => a+b, 0);
const avg = total / amounts.length;
return [{ json: { total, avg, count: items.length, max: Math.max(...amounts) }}];

// 3. Gọi thư viện built-in của n8n
const crypto = require('crypto');
const hash = crypto.createHash('sha256')
  .update($input.first().json.email)
  .digest('hex');
return [{ json: { hash } }];

version: '3.8'
services:
  n8n:
    image: n8nio/n8n:latest

      - DB_POSTGRESDB_DATABASE=n8n

      - GENERIC_TIMEZONE=Asia/Ho_Chi_Minh

      - n8n_data:/home/node/.n8n

      postgres:

Checklist bảo mật trước khi go-live
🔒
Credentials & Secrets
Dùng biến môi trường cho tất cả API keys. Không hardcode trong workflow. Bật encryption key mạnh.
🌐
HTTPS & Reverse Proxy
Dùng Nginx/Caddy làm reverse proxy với SSL/TLS. Tắt HTTP redirect hoàn toàn.
👤
User Management
Tạo tài khoản riêng cho từng thành viên. Không dùng chung 1 account admin.
💾
Backup tự động
Cron backup PostgreSQL hàng ngày. Export workflows định kỳ ra Git repository.
