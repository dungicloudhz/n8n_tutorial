Phần 1 — HTML

HTML5

HTML là gì?

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu để tạo cấu trúc trang web. Trình duyệt đọc HTML và hiển thị nội dung. HTML không phải ngôn ngữ lập trình — nó định nghĩa cái gì xuất hiện trên trang.

🔗

📦

Box Model — Khái niệm quan trọng nhất!

Mọi phần tử HTML đều là một "hộp" với 4 lớp lồng nhau:

margin (khoảng ngoài)
border (viền)
padding (khoảng trong)
content (nội dung)
